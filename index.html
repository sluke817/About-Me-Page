CS 2050

Alogrithm Design and Programming II in C

CS 2050 is one of the first serious programming classes. CS 2050 is the second course in the Alogrithm and Design class arc. CS 2050 taught higher level code design patterns and important data structes such as heaps, stacks, and trees.

Official Description from the University of Missouri website: "A study of fundamental techniques and algorithms for representing and manipulating data structures. Topics include data abstraction, recursion, stacks, queues, linked lists, trees, efficient methods of sorting and searching, and Big-O analysis."

CS 3050

Advanced Algorithm Design

CS 3050 is one of the most important classes in the CS track. CS 3050 teaches the applicable usage of complex data structes such as search trees and graphs. The mathmatical proofs behind these major concepts is taught in depth and there is an emphasis on the efficiency behind these data structures.

"Official Description from the University of Missouri website: This class surveys fundamental algorithms and data structures that have wide practical applicability, including search trees and graph algorithms. Emphasis is placed on techniques for efficient implementation and good software development methodologies."
